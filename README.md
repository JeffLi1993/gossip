# Gossip

[Hompage](https://syhily.github.io/gossip) | [首页](https://syhily.github.io/gossip)

A comment system for Ghost platform and other static site generator. Support [duoshuo](http://duoshuo.com), [Disqus](https://disqus.com) and [Wordpress](https://wordpress.org) export file. And it's under heavy development now.
